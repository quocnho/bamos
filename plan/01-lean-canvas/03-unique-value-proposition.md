# Unique Value Proposition

## Current State
**"BamOS — NixOS Distribution for the Vietnamese Community"**
**"Cài xong là dùng" (Out-of-the-box. Plug & Play.)**

BamOS là bản phân phối NixOS **duy nhất** dành cho người Việt, kết hợp sức mạnh của:
- **Declarative + Reproducible**: Mọi cấu hình qua code Nix, version-control bằng Git
- **Immutable + Rollback**: Hệ thống không thể bị phá hỏng, rollback tức thì qua reboot
- **Bản địa hóa OOTB**: Tiếng Việt hoạt động ngay sau cài đặt, không cần terminal
- **12 ISO variants**: 3 DE × 4 Editions phục vụ mọi nhu cầu
