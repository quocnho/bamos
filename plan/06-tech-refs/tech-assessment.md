## Key Decisions

### Why NixOS over Ubuntu/Fedora?
- **Reproducibility**: Mọi cài đặt từ cùng git commit giống hệt nhau
- **Immutability**: Hệ thống gốc không thể bị phá hỏng
- **Rollback**: Khôi phục về generation cũ trong < 2 phút
- **Declarative**: Toàn bộ hệ thống được quản lý qua code, version-controlled

### Why flake-parts over raw flakes?
- Module system giống NixOS module system — quen thuộc
- `perSystem` pattern giúp tổ chức code sạch hơn
- Hỗ trợ `imports` để modular hóa host definitions

### Why Zen kernel over LTS?
- Desktop-optimized scheduler (better responsiveness)
- Low-latency audio (quan trọng cho Studio Edition)
- Gaming performance (quan trọng cho Gaming Edition)
- Tương thích rộng với phần cứng Intel + AMD

### Why 3 DEs instead of 1?
- GNOME: Đơn giản, touch-friendly → người dùng phổ thông
- KDE Plasma: Tùy biến cao → power users, multi-monitor
- COSMIC: Rust-based, tiling → early adopters, developers
