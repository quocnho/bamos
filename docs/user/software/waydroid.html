📱 Waydroid — Ứng dụng Android trên BamOS

Waydroid cho phép bạn chạy ứng dụng Android ngay trên màn hình BamOS, sử dụng công nghệ container dựa trên LXC (Linux Containers). Không cần máy ảo — ứng dụng Android chạy mượt mà như ứng dụng native.

📹 Chạy ứng dụng Android với Waydroid trên BamOS (sẽ thêm sau)

💡 Waydroid hoạt động thế nào?

Waydroid chạy một container Android đầy đủ (dựa trên Android Open Source Project) và tích hợp trực tiếp với hệ thống Wayland của BamOS. Ứng dụng Android xuất hiện trong cửa sổ riêng, có thể ghim vào taskbar và chuyển đổi qua lại như ứng dụng thông thường.

⚠️ Yêu cầu

Waydroid yêu cầu GPU có hỗ trợ đồ họa (Intel, AMD, NVIDIA). Cần khoảng 4GB RAM trống và 5GB dung lượng ổ cứng cho image Android. Chạy tốt nhất trên BamOS Gaming Edition nhờ kernel XanMod.

📥 Cài đặt Waydroid

BamOS đã bao gồm Waydroid trong hệ thống. Nếu chưa cài, bạn có thể cài như sau:

# Cài Waydroid
sudo rpm-ostree install waydroid

# Khởi động lại hệ thống
sudo systemctl reboot

# Hoặc cài qua script (cho các bản không có sẵn)
# curl -sf https://raw.githubusercontent.com/waydroid/waydroid/main/scripts/install.sh | sudo bash

⚙ Cấu hình ban đầu

# Khởi tạo Waydroid (tải Android image ~2GB)
sudo waydroid init

# Nếu muốn dùng image với Google Play
sudo waydroid init -s GAPPS

# Khởi động Waydroid session
waydroid session start &

⏳ Thời gian tải

Lần đầu chạy sudo waydroid init sẽ tải Android image khoảng 2GB. Quá trình này phụ thuộc vào tốc độ internet của bạn.

🎯 Sử dụng Waydroid

Mở ứng dụng Android

# Mở Waydroid container
waydroid show-full-ui

# Cài APK
waydroid install ~/Downloads/my-app.apk

# Mở ứng dụng từ terminal
waydroid app launch com.example.app

# Liệt kê ứng dụng đã cài
waydroid app list

Tương tác với ứng dụng

  • 🖱 Chuột: Click và kéo thả như bình thường
  • ⌨ Bàn phím: Gõ trực tiếp
  • 📱 Cảm ứng: Màn hình cảm ứng được hỗ trợ
  • 🔄 Xoay màn hình: Ctrl + F11
  • 🏠 Home: Ctrl + F1
  • ↩ Back: Ctrl + F2
  • 📋 Recent: Ctrl + F3

📱 Google Play Services

Để có Google Play trên Waydroid, bạn cần khởi tạo với tùy chọn -s GAPPS:

# Nếu chưa khởi tạo
sudo waydroid init -s GAPPS

# Nếu đã khởi tạo, cần làm lại từ đầu
sudo waydroid session stop
sudo waydroid container stop
sudo rm -rf /var/lib/waydroid
sudo waydroid init -s GAPPS
waydroid session start &

⚠️ Lưu ý về Google Play

Google Play trên Waydroid sử dụng giả lập, có thể gặp một số hạn chế. Một số ứng dụng có DRM (Netflix, Disney+) sẽ không hoạt động. Các ứng dụng thông thường (Facebook, Messenger, Zalo, TikTok) hoạt động tốt.

🔧 Cấu hình nâng cao

Cấu hình Waydroid

File cấu hình Waydroid nằm tại /etc/waydroid/cfg/waydroid.cfg và ~/.local/share/waydroid/cfg/waydroid.cfg:

# Ví dụ cấu hình
[waydroid]
width = 1920
height = 1080
dpi = 240

[properties]
ro.sf.lcd_density = 240
persist.waydroid.multi_windows = true

Tăng dung lượng

# Mặc định image Android là 4GB. Để tăng:
sudo waydroid expand 8G

📱 Zalo 🟢 Tốt Nhắn tin, gọi điện
📱 Messenger 🟢 Tốt Có thể dùng bản web thay thế
📱 TikTok 🟢 Tốt Xem và đăng video
📱 Grab / Gojek 🟢 Tốt Đặt xe, giao hàng
📱 Banking apps 🟡 Trung bình Một số app có thể không hoạt động
📱 Netflix / Disney+ 🔴 Không DRM không hoạt động

🛠 Xử lý sự cố

📝 Waydroid không khởi động

# Kiểm tra trạng thái
sudo waydroid status

# Kiểm tra log
sudo journalctl -u waydroid-container -f

# Kiểm tra binder module
lsmod | grep binder

# Nếu thiếu binder, load module
sudo modprobe binder_linux

📝 Ứng dụng không hiển thị

# Khởi động lại Waydroid
waydroid session stop
waydroid session start &

# Clear cache
waydroid session stop
sudo rm -rf /var/lib/waydroid/cache
waydroid session start &
