🗜 AppImage trên BamOS

AppImage là định dạng đóng gói ứng dụng portable trên Linux. Một file AppImage duy nhất chứa toàn bộ ứng dụng và dependencies — bạn chỉ cần tải về, cấp quyền chạy, và sử dụng. Không cần cài đặt, không cần root.

📹 Hướng dẫn sử dụng AppImage trên BamOS (sẽ thêm sau)

💡 Ưu điểm của AppImage

✓ Portable — mang đi đâu cũng được
✓ Không cần cài đặt — chạy ngay sau khi tải
✓ Không phụ thuộc vào hệ thống — tự chứa dependencies
✓ Tương thích mọi bản phân phối Linux
✓ Cô lập — không ảnh hưởng đến hệ thống

🚀 Cách sử dụng AppImage

Cách thủ công (cơ bản)

# Tải AppImage (ví dụ: Krita)
wget https://download.kde.org/stable/krita/5.2/krita-5.2.0-x86_64.appimage

# Cấp quyền chạy
chmod +x krita-*.appimage

# Chạy
./krita-*.appimage

Tích hợp vào menu ứng dụng

Có hai cách để tích hợp AppImage vào menu:

Cách 1: Gear Lever (Khuyên dùng)

Gear Lever là ứng dụng GUI giúp quản lý AppImage một cách trực quan. Nó tự động tích hợp AppImage vào menu ứng dụng của BamOS.

# Cài Gear Lever
flatpak install flathub it.mijorus.gearlever

Sau khi cài Gear Lever:

  1. Mở Gear Lever từ menu ứng dụng
  2. Kéo thả file .appimage vào cửa sổ Gear Lever
  3. Nhấn "Integrate" — AppImage xuất hiện ngay trong menu!

Cách 2: AppImageLauncher

# Tải AppImageLauncher từ trang chủ
# Hoặc cài qua distrobox
distrobox enter ubuntu
sudo add-apt-repository ppa:appimagelauncher-team/stable
sudo apt update
sudo apt install appimagelauncher

🔧 Gear Lever (Chi tiết)

Gear Lever là công cụ quản lý AppImage hàng đầu cho Linux. Ngoài tích hợp vào menu, Gear Lever còn hỗ trợ:

Tính năng Mô tả
➕ Tích hợp AppImage Thêm AppImage vào menu ứng dụng
🗑 Gỡ tích hợp Xóa AppImage khỏi menu
🔄 Cập nhật Kiểm tra và cập nhật AppImage
🔍 Tìm kiếm Tìm AppImage trên các kho
⚙ Cấu hình Tùy chỉnh hành vi mặc định

📝 Mẹo Gear Lever

Gear Lever cho phép bạn chọn thư mục lưu trữ AppImage mặc định (ví dụ: ~/Applications). Tất cả AppImage trong thư mục này sẽ tự động được quét và hiển thị.

⌨ Quản lý AppImage qua CLI

# Tạo thư mục chứa AppImage
mkdir -p ~/Applications

# Di chuyển AppImage vào thư mục
mv ~/Downloads/*.appimage ~/Applications/

# Cấp quyền cho tất cả
chmod +x ~/Applications/*.appimage

# Tạo alias để chạy nhanh
echo 'alias obs="~/Applications/OBS-Studio-*.appimage"' >> ~/.bashrc

Ứng dụng Tải tại Ghi chú

🛠 Xử lý sự cố

⚠️ AppImage không chạy

# Kiểm tra FUSE
sudo dnf install fuse3 fuse3-libs

# Kiểm tra quyền
ls -la ~/Applications/*.appimage

# Chạy với verbose
./app.AppImage --verbose

# Chạy với FUSE debug
APPIMAGE_EXTRACT_AND_RUN=1 ./app.AppImage

# Cài FUSE nếu thiếu
sudo rpm-ostree install fuse3

📝 Mẹo

Một số AppImage có thể không tương thích với kernel XanMod trên Gaming Edition. Hãy thử dùng Flatpak hoặc Distrobox như giải pháp thay thế.
