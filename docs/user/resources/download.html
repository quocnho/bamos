⬇ Tải xuống BamOS

Chào mừng bạn đến với trang tải xuống BamOS! Dưới đây là các phiên bản ISO mới nhất. Hãy chọn edition phù hợp với nhu cầu của bạn.

Checksum & Xác minh

Sau khi tải ISO, bạn nên xác minh tính toàn vẹn của tệp bằng checksum để đảm bảo tệp không bị hỏng hoặc bị thay đổi.

SHA256 Checksums

# Desktop sha256sum bamos-desktop-0.4.0.iso # f7c3b... (xem trên trang releases) # Gaming sha256sum bamos-gaming-0.4.0.iso # a1b2c... (xem trên trang releases) # Server sha256sum bamos-server-0.4.0.iso # d4e5f... (xem trên trang releases) # Minimal sha256sum bamos-minimal-0.4.0.iso # g6h7i... (xem trên trang releases)

Cách xác minh trên Linux / macOS

# Tải file checksum curl -LO https://github.com/quocnho/bamos/releases/latest/download/sha256sums.txt # Xác minh sha256sum -c sha256sums.txt --ignore-missing # Kết quả: bamos-desktop-0.4.0.iso: OK

Cách xác minh trên Windows (PowerShell)

# Tính checksum Get-FileHash .\bamos-desktop-0.4.0.iso -Algorithm SHA256 # So sánh với giá trị trên trang releases
🔒 Tại sao cần xác minh? Checksum giúp bạn phát hiện tệp bị lỗi trong quá trình tải hoặc bị can thiệp trái phép. Luôn xác minh trước khi cài đặt!

Hướng dẫn ghi ISO vào USB

Sau khi tải ISO, bạn cần ghi vào USB để boot và cài đặt:

Linux

# Xác định ổ USB (ví dụ /dev/sdb) lsblk # Ghi ISO sudo dd if=bamos-desktop-0.4.0.iso of=/dev/sdX bs=4M status=progress # Hoặc dùng balenaEtcher sudo balena-etcher-electron

Windows

  • Rufus — Công cụ ghi ISO nhanh và dễ dàng
  • balenaEtcher — Đa nền tảng, đơn giản
  • Ventoy — Boot nhiều ISO từ một USB

macOS

# Ghi ISO sudo dd if=bamos-desktop-0.4.0.iso of=/dev/rdisk2 bs=4M status=progress # Hoặc dùng balenaEtcher
