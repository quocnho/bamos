🔧 Thông số kỹ thuật BamOS

Trang này cung cấp thông tin chi tiết về yêu cầu phần cứng, kích thước ISO, và thông số kỹ thuật của từng phiên bản BamOS.

Yêu cầu phần cứng tối thiểu

Thành phần Tối thiểu Khuyến nghị
CPU x86_64, 2 nhân 4 nhân trở lên
RAM 4 GB 8 GB (Desktop/Gaming)
2 GB (Minimal/Server)
Ổ cứng 20 GB 64 GB SSD trở lên
GPU Tích hợp Intel/AMD NVIDIA/AMD rời cho Gaming
Kết nối mạng Có dây hoặc WiFi Băng thông rộng để tải ISO & cập nhật
USB 8 GB 16 GB (dùng ghi ISO)
💡 Lưu ý: BamOS sử dụng OSTree nên cần thêm dung lượng cho các deployment cũ. Khuyến nghị ổ cứng ≥ 64 GB.

Kích thước ISO

Edition Kích thước ISO Dung lượng sau cài đặt
🖥 Desktop ~3.2 GB ~12 GB
🎮 Gaming ~5.8 GB ~20 GB
🗄 Server ~1.5 GB ~5 GB
⚡ Minimal ~1.2 GB ~4 GB

Thông số kỹ thuật theo phiên bản

Bộ phận Desktop Gaming Server Minimal
Desktop Environment GNOME 47 + COSMIC GNOME 47 Gaming Mode CLI (không DE) CLI (không DE)
Kernel mặc định Zen Zen + XanMod LTS LTS
Display Server Wayland Wayland + XWayland Không có Không có
Sound System PipeWire PipeWire PipeWire PipeWire
Package Manager Nix + Flatpak Nix + Flatpak Nix Nix
Init System systemd systemd systemd systemd
Filesystem Btrfs (mặc định) Btrfs (mặc định) Btrfs / Ext4 Btrfs / Ext4
Bootloader GRUB GRUB GRUB / systemd-boot GRUB / systemd-boot

Hỗ trợ kiến trúc

Kiến trúc Trạng thái Ghi chú
x86_64 (AMD64) ✅ Chính thức Desktop & Server chính
aarch64 (ARM64) 🔄 Đang phát triển Raspberry Pi, Apple Silicon (Asahi)

Hỗ trợ GPU

Hãng GPU Driver Ghi chú
Intel i915 (open-source) Có sẵn, hoạt động tốt
AMD amdgpu (open-source) Có sẵn, hỗ trợ Vulkan tốt
NVIDIA nvidia-open (open kernel module) Cài thêm qua nixpkgs.nvidia
