🧠 Kernel trên BamOS

BamOS sử dụng chiến lược đa kernel (dual-kernel) để tối ưu cho từng mục đích sử dụng. Tuỳ theo phiên bản, bạn sẽ được cài sẵn kernel phù hợp nhất.

💡 Bạn muốn biết mình đang dùng kernel nào? Gõ lệnh: uname -r

Chiến lược kernel của BamOS

BamOS cung cấp 3 loại kernel chính, mỗi loại phục vụ một mục đích khác nhau:

Kernel Sử dụng cho Đặc điểm
🐢 LTS (Long-Term Support) ISO cài đặt, Server, Minimal Ổn định nhất, ít thay đổi, hỗ trợ lâu dài
⚡ Zen Kernel Desktop, Gaming (mặc định) Cân bằng hiệu năng và ổn định, tối ưu cho desktop
🚀 XanMod Kernel Gaming (tùy chọn) Tối đa hiệu năng chơi game, giảm latency

🐢 LTS Kernel — Cho ổn định tối đa

Kernel LTS (Long-Term Support) được sử dụng trong ISO cài đặt BamOS để đảm bảo khả năng tương thích cao nhất với nhiều loại phần cứng khác nhau. Nó cũng là kernel mặc định cho Server và Minimal edition.

Đặc điểm

  • Dựa trên nhánh kernel ổn định nhất của Linux
  • Được bảo trì trong nhiều năm với các bản vá bảo mật
  • Ít thay đổi đột ngột, giảm nguy cơ hỏng tương thích
  • Phù hợp với máy chủ, máy tính văn phòng, người dùng cần ổn định

⚡ Zen Kernel — Cân bằng cho Desktop

Zen Kernel là kernel mặc định cho BamOS Desktop và Gaming Edition. Nó được tối ưu cho hiệu năng sử dụng hàng ngày, mang lại trải nghiệm mượt mà hơn so với kernel chính thống.

Đặc điểm

  • Dựa trên nhánh kernel mainline mới nhất
  • Tối ưu scheduler cho tương tác desktop (MuQSS)
  • Giảm độ trễ (latency) cho ứng dụng thời gian thực
  • Cải thiện thời gian phản hồi khi đa nhiệm
  • Tích hợp sẵn nhiều driver và tính năng mới
# Kiểm tra phiên bản Zen Kernel uname -r # Kết quả sẽ có "zen" ở cuối, vd: 6.8.7-zen1-1-zen

🚀 XanMod Kernel — Tối đa cho Gaming

XanMod Kernel là kernel hiệu năng cao dành riêng cho gaming, có sẵn trên BamOS Gaming Edition. Nó được tinh chỉnh để giảm tối đa độ trễ và tăng throughput.

Đặc điểm

  • Tối ưu cho gaming với độ trễ cực thấp
  • Sử dụng BORE scheduler thay vì CFS
  • Tích hợp các bản vá từ Liquorix
  • Hỗ trợ TCP BBR cho kết nối mạng nhanh hơn
  • Các bản vá âm thanh cho trải nghiệm chơi game tốt hơn

Cài đặt XanMod trên Gaming Edition

# Cài đặt XanMod Kernel sudo nix profile install nixpkgs#linux_xanmod # Kiểm tra sau khi cài uname -r # Kết quả: 6.8.7-xanmod1

So sánh chi tiết

Tiêu chí LTS Zen XanMod
Ổn định ⭐⭐⭐⭐⭐ ⭐⭐⭐⭐ ⭐⭐⭐
Hiệu năng Desktop ⭐⭐⭐ ⭐⭐⭐⭐⭐ ⭐⭐⭐⭐⭐
Hiệu năng Gaming ⭐⭐ ⭐⭐⭐⭐ ⭐⭐⭐⭐⭐
Độ trễ (latency) Thấp Rất thấp Cực thấp
Tương thích phần cứng Rất cao Cao Trung bình
Tần suất cập nhật Thấp (vá bảo mật) Cao (theo mainline) Cao (theo mainline)

Chuyển đổi kernel

Bạn có thể chuyển đổi kernel bất kỳ lúc nào:

# Chuyển sang Zen sudo nix profile install nixpkgs#linux_zen # Chuyển sang LTS sudo nix profile install nixpkgs#linux_lts # Chuyển sang XanMod sudo nix profile install nixpkgs#linux_xanmod

Sau khi cài đặt, khởi động lại máy và chọn kernel mới trong GRUB.

⚠️ Lưu ý: Khi chuyển kernel, một số driver hoặc module có thể không tương thích. Luôn giữ lại kernel cũ trong GRUB để rollback nếu cần.

Kernel parameters

Bạn có thể chỉnh sửa kernel parameters khi boot để gỡ rối hoặc tối ưu:

# Tham số thường dùng: nomodeset # Bỏ qua driver GPU (khi màn hình đen) mitigations=off # Tắt các bản vá bảo mật CPU (tăng hiệu năng) acpi=off # Tắt ACPI (khi không boot được) quiet # Ẩn log khi boot systemd.debug-shell=1 # Mở shell debug sớm
