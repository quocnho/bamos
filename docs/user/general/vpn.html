🔒 Cấu Hình VPN trên BamOS

BamOS hỗ trợ nhiều giao thức VPN khác nhau. Dưới đây là hướng dẫn cấu hình WireGuard, OpenVPN và ZeroTier.

WireGuard (Khuyến nghị)

WireGuard là giao thức VPN hiện đại, nhanh và bảo mật. Đây là lựa chọn khuyến nghị trên BamOS.

Cài đặt qua NixOS Config

{ config, pkgs, ... }: {
  networking.wireguard.interfaces = {
    wg0 = {
      ips = [ "10.0.0.2/24" ];
      privateKey = "your-private-key-here";

      peers = [
        {
          publicKey = "peer-public-key";
          allowedIPs = [ "0.0.0.0/0" "::/0" ];
          endpoint = "vpn.example.com:51820";
          persistentKeepalive = 25;
        }
      ];
    };
  };
}

Kết nối thủ công

# Tạo key pair
wg genkey | tee privatekey | wg pubkey > publickey

# Cấu hình file /etc/wireguard/wg0.conf
sudo nano /etc/wireguard/wg0.conf

# Bật kết nối
sudo wg-quick up wg0

# Kiểm tra trạng thái
sudo wg show

# Tắt kết nối
sudo wg-quick down wg0

💡 Mẹo

WireGuard hoạt động tốt với NetworkManager. Bạn có thể import file .conf qua GUI NetworkManager để quản lý kết nối dễ dàng hơn.

OpenVPN

OpenVPN vẫn được nhiều nhà cung cấp dịch vụ hỗ trợ.

Cài đặt gói

# Thêm vào configuration.nix
{ config, pkgs, ... }: {
  environment.systemPackages = with pkgs; [
    openvpn
    networkmanager-openvpn
  ];
  networking.networkmanager.enable = true;
}

Import cấu hình

# Qua NetworkManager GUI
# Hoặc dùng CLI:
sudo nmcli connection import type openvpn file /path/to/config.ovpn

Kết nối thủ công

# Kết nối
sudo openvpn --config /path/to/config.ovpn

# Kết nối với auth file
sudo openvpn --config /path/to/config.ovpn --auth-user-pass /path/to/auth.txt

⚠️ Lưu ý

Nếu bạn dùng NetworkManager, hãy đảm bảo dịch vụ networkmanager-openvpn đã được bật. Trên NixOS, thêm networkmanager-openvpn vào networking.networkmanager.plugins.

ZeroTier

ZeroTier tạo mạng mesh riêng tư, rất hữu ích cho kết nối giữa các thiết bị.

Cài đặt

{ config, pkgs, ... }: {
  services.zerotierone.enable = true;
  services.zerotierone.joinNetworks = [ "your-network-id" ];
}

Quản lý ZeroTier

# Kiểm tra trạng thái
sudo zerotier-cli status

# Join mạng
sudo zerotier-cli join <network-id>

# Leave mạng
sudo zerotier-cli leave <network-id>

# Liệt kê mạng đã join
sudo zerotier-cli listnetworks

# Hiển thị thông tin node
sudo zerotier-cli info

Kiểm Tra Kết Nối VPN

# Kiểm tra địa chỉ IP
curl ifconfig.me

# Kiểm tra DNS leak
curl https://ipleak.net

# Kiểm tra kết nối
ping -c 4 10.0.0.1  # IP nội bộ của VPN

✅ VPN đã hoạt động

Khi VPN kết nối thành công, bạn sẽ thấy địa chỉ IP thay đổi và có thể truy cập tài nguyên nội bộ qua mạng VPN.
