⚙ Tinh Chỉnh Hệ Thống BamOS

BamOS được thiết kế để hoạt động tốt ngay sau khi cài đặt. Tuy nhiên, bạn có thể tinh chỉnh thêm để phù hợp với nhu cầu cá nhân. Dưới đây là các tùy chỉnh phổ biến.

Giao Diện & Theme

Đổi Theme GNOME

BamOS GNOME Edition sử dụng theme mặc định của GNOME. Bạn có thể đổi theme bằng GNOME Tweaks:

💡 Cài GNOME Tweaks

# Cài GNOME Tweaks qua Flatpak
flatpak install flathub org.gnome.tweaks

# Hoặc cài theme qua GNOME Look
flatpak install flathub org.gnome.Loupe

Đổi Theme KDE Plasma

KDE Plasma có sẵn công cụ System Settings → Appearance. Bạn có thể tải thêm theme từ KDE Store.

Font Chữ

BamOS đã cấu hình sẵn font cho tiếng Việt. Bạn có thể đổi font mặc định:

GNOME:

gsettings set org.gnome.desktop.interface font-name 'Noto Sans 11'
gsettings set org.gnome.desktop.interface monospace-font-name 'JetBrains Mono 11'

KDE Plasma: Vào System Settings → Fonts để thay đổi trực quan.

GNOME Extensions

GNOME Edition của BamOS đi kèm một số extension hữu ích. Bạn có thể cài thêm:

ExtensionCông dụngCài đặt
Dash to Dock Thanh dock giống macOS flatpak install ... gnome-shell-extensions
Blur my Shell Hiệu ứng mờ cho GNOME Shell GNOME Extensions website
Vitals CPU/RAM/Disk trên panel GNOME Extensions website
AppIndicator Hỗ trợ system tray icons Có sẵn trong BamOS

⚠️ Lưu ý trên NixOS

Trên NixOS, extension GNOME có thể bị ghi đè khi update. Để cố định extension, hãy thêm vào cấu hình NixOS của bạn:

{ config, pkgs, ... }: {
  services.xserver.desktopManager.gnome = {
    enable = true;
    # Thêm extension vào đây
    extraExtensions = with pkgs.gnomeExtensions; [
      dash-to-dock
      blur-my-shell
    ];
  };
}

Tinh Chỉnh Hiệu Suất

Tắt Animations

Tắt hiệu ứng hoạt ảnh để tiết kiệm tài nguyên:

# GNOME
gsettings set org.gnome.desktop.interface enable-animations false

# KDE Plasma: System Settings → Workspace Behavior → Desktop Effects

ZRAM

BamOS đã bật ZRAM mặc định — nén RAM để tăng dung lượng khả dụng. Kiểm tra trạng thái:

zramctl

CPU Governor

Điều chỉnh chế độ CPU cho hiệu năng hoặc tiết kiệm pin:

# Xem governor hiện tại
cat /sys/devices/system/cpu/cpu0/cpufreq/scaling_governor

# Đặt performance (mặc định)
echo performance | sudo tee /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor

# Đặt powersave (tiết kiệm pin)
echo powersave | sudo tee /sys/devices/system/cpu/cpu*/cpufreq/scaling_governor

Preload

BamOS Gaming và Developers Edition có sẵn bleachbit và preload. Cài thêm preload để tăng tốc mở ứng dụng:

# Thêm vào configuration.nix
{ config, pkgs, ... }: {
  services.preload.enable = true;
}

Tinh Chỉnh NixOS (Nâng Cao)

BamOS dựa trên NixOS, bạn có thể tùy biến mọi thứ qua file cấu hình. Xem thêm tại 🛠 Custom Image.

📝 Ghi chú

Sau khi chỉnh sửa cấu hình NixOS, chạy sudo nixos-rebuild switch để áp dụng thay đổi. Nếu có lỗi, bạn có thể rollback bằng cách chọn generation cũ trong GRUB.
