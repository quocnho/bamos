🚀 Game Launcher trên BamOS

BamOS hỗ trợ đa dạng các game launcher, giúp bạn truy cập kho game từ nhiều nền tảng khác nhau. Dưới đây là danh sách các launcher phổ biến và cách sử dụng chúng.

📹 Hướng dẫn sử dụng các game launcher trên BamOS (sẽ thêm sau)

🎯 1. Steam

Steam là game launcher chính và được cài đặt sẵn trên Gaming Edition. Với hàng ngàn tựa game native Linux và hàng chục ngàn game Windows chạy qua Proton, Steam là trung tâm giải trí không thể thiếu.

💡 Mẹo

Kích hoạt Steam Play (Proton) cho tất cả game: Steam → Settings → Compatibility → ☑ Enable Steam Play for all other titles.

# Cài Steam (nếu chưa có)
sudo rpm-ostree install steam

# Hoặc qua Flatpak
flatpak install flathub com.valvesoftware.Steam

🎮 2. Lutris

Lutris là trình quản lý game mã nguồn mở, hỗ trợ tích hợp nhiều nền tảng: Steam, GOG, Epic Games, Humble Bundle, và các bản cài đặt thủ công. Lutris tự động cấu hình Wine/Proton cho từng game.

# Cài Lutris
sudo rpm-ostree install lutris

# Hoặc Flatpak
flatpak install flathub net.lutris.Lutris

📝 Lutris Runners

Lutris hỗ trợ nhiều runner: Wine (vanilla, GE, staging), Proton, Steam, RetroArch, DOSBox, ScummVM, MAME và nhiều hơn nữa.

🦸 3. Heroic Games Launcher

Heroic Games Launcher là launcher mã nguồn mở cho Epic Games Store, GOG và Amazon Games. Không cần cài đặt client Windows cồng kềnh — Heroic chạy native trên Linux.

Tính năng chính

  • Đăng nhập Epic Games, GOG, Amazon
  • Tải và cài game trực tiếp
  • Tích hợp Wine, Proton, GameScope, MangoHud
  • Cloud saves
  • Quản lý DLC
# Cài Heroic Games Launcher
flatpak install flathub com.heroicgameslauncher.hgl

💡 Mẹo

Trong Heroic, bạn có thể chọn Wine/Proton riêng cho từng game, bật GameScope và MangoHud ngay trong cài đặt của từng game.

🍾 4. Bottles

Bottles là công cụ quản lý tiền tố Wine (Wine prefixes) với giao diện đồ họa đẹp mắt. Mỗi "bottle" là một môi trường Windows riêng biệt — lý tưởng để chạy game và ứng dụng Windows.

# Cài Bottles
flatpak install flathub com.usebottles.bottles

Các loại Bottle

Loại Mục đích Cấu hình
🎮 Gaming Chơi game DXVK, VKD3D, GameMode, MangoHud
🖥 Software Ứng dụng văn phòng WinHTTP, .NET, fonts
⚙ Custom Tùy chỉnh thủ công Toàn quyền kiểm soát

🕹️ 5. itch.io

Trình quản lý game cho itch.io — nơi hàng ngàn indie game được phát hành. Hỗ trợ tải, cập nhật và chạy game trực tiếp.

# Cài itch.io App
flatpak install flathub io.itch.itch

⚠️ Lưu ý về Anti-Cheat

Một số launcher và game có Anti-Cheat (EAC, BattlEye) cần cấu hình đặc biệt. Xem thêm tại trang Tùy chọn khởi chạy.
