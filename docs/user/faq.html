❓ Câu hỏi thường gặp (FAQ)

📥 Cài đặt

BamOS có miễn phí không?

Có! BamOS hoàn toàn miễn phí, mã nguồn mở (MIT License). Bạn có thể tải, cài đặt, sửa đổi và phân phối lại.

Làm sao để tải BamOS?

Truy cập trang Download để tải ISO mới nhất. Chọn phiên bản phù hợp với nhu cầu của bạn.

Cấu hình tối thiểu để chạy BamOS là gì?

RAM: 4GB (khuyên dùng 8GB), Ổ cứng: 30GB, CPU: x86_64, UEFI. Xem thêm tại Specs.

Tôi có thể cài BamOS song song với Windows không?

Có. Trình cài đặt Calamares hỗ trợ dual-boot. Bạn nên thu nhỏ partition Windows trước trong Disk Management. Xem phương pháp thay thế.

BamOS có hỗ trợ Secure Boot không?

Hiện tại BamOS chưa hỗ trợ Secure Boot. Bạn cần tắt Secure Boot trong BIOS để boot BamOS.

🖥️ Sử dụng

Làm sao để gõ tiếng Việt?

Nhấn Ctrl+Space để chuyển đổi giữa tiếng Anh và tiếng Việt. Hỗ trợ Telex, VNI, VIQR. Xem Tweaks.

Làm sao để cài ứng dụng?

Mở Software Center (GNOME Software / KDE Discover) hoặc dùng Flatpak: flatpak install flathub ten-ung-dung. Xem Software Center.

Tôi có thể chơi game trên BamOS không?

Có! BamOS Gaming Edition tích hợp sẵn Steam, Lutris, GameScope và MangoHud. Hỗ trợ Proton để chơi game Windows. Xem Gaming.

Cập nhật BamOS như thế nào?

Dùng lệnh: sudo nixos-rebuild switch --flake . hoặc qua Software Center. Xem Updates.

🔧 Kỹ thuật

BamOS dùng kernel gì?

BamOS dùng chiến lược đa kernel: LTS cho ISO live, Zen cho mặc định, XanMod cho Gaming edition. Xem Kernel.

Làm sao để tự build ISO BamOS?

Fork GitHub repo, sửa flake.nix, chạy: nix build .#iso-gnome-standard. Xem Custom Image.

Có binary cache không?

Có. BamOS dùng Cachix để cache bản build, giúp cập nhật nhanh hơn. Xem Cache.

👥 Cộng đồng

Làm sao để đóng góp cho BamOS?

Fork repo trên GitHub, tạo nhánh, commit, gửi Pull Request. Xem Contributing.

Làm sao để báo lỗi?

Mở issue trên GitHub. Cung cấp log, mô tả chi tiết và cách tái hiện. Xem Bugs.

Chưa tìm thấy câu trả lời?

Tham gia cộng đồng BamOS trên Discord, Facebook Group để được hỗ trợ.
