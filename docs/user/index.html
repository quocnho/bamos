Tại sao chọn BamOS?

Kết hợp tinh hoa công nghệ NixOS và triết lý Việt Nam.

🎋

Triết lý Tre Trăm Đốt

Module độc lập, liên kết chặt chẽ. Gãy một đốt, cây vẫn vững — giống như BamOS: modular, minh bạch, tái tạo được.

🇻🇳

Cho người Việt Nam

Giao diện tiếng Việt toàn diện. Từ lập trình viên đến người dùng phổ thông — BamOS nói ngôn ngữ của bạn.

🔒

Ổn định & An toàn

Dựa trên NixOS — mỗi thay đổi được cô lập. Rollback về phiên bản trước chỉ trong vài giây.

⚡

Khai báo & Tái tạo

Cấu hình hệ thống bằng code. Chia sẻ môi trường dev chỉ với một file. "It works on my machine" không còn là vấn đề.

Sức mạnh NixOS trong BamOS

BamOS kế thừa toàn bộ sức mạnh của NixOS — hệ điều hành khai báo (declarative) hàng đầu thế giới.

🛠️

Môi trường dev luôn sẵn sàng

Không còn "trên máy tôi chạy được mà". Môi trường phát triển đồng nhất, tái tạo chính xác trên mọi máy.

🌐

Một công cụ, mọi ngôn ngữ

Dù bạn dùng Python, Rust, Node.js hay Go — tất cả cùng chung một cách build, một cách deploy.

☁️

Từ code đến cloud

Build Docker image, Amazon EC2, Google Cloud — tất cả từ một file cấu hình duy nhất. DevOps chưa bao giờ dễ hơn.

⏪

Rollback bất cứ lúc nào

Nix không bao giờ ghi đè file. Mỗi lần cập nhật là một phiên bản mới — rollback chỉ trong tích tắc.

🧪

Thử nghiệm không sợ hãi

Cài phần mềm mới, thử nghiệm bleeding-edge — không lo hệ thống bị hỏng. Môi trường làm việc luôn nguyên vẹn.

🤝

Cộng đồng là sức mạnh

Đóng góp chỉ với một Pull Request. Cộng đồng toàn cầu sẵn sàng review, hướng dẫn, và cùng bạn xây dựng.

Khai báo. Tái tạo. Linh hoạt.

BamOS dùng Nix Flakes — chuẩn hiện đại để khai báo toàn bộ hệ thống. Một file, mọi thứ.

🎯 flake.nix — Toàn bộ hệ thống

Với Flakes, mọi thứ được khai báo trong flake.nix. Từ kernel, desktop, driver NVIDIA, đến từng package — tất cả đều tái tạo chính xác trên mọi máy.

flake.nix — BamOS RỒNG TINH VÂN
description = "Hệ điều hành cho người Việt";

outputs = { self, nixpkgs, bamos, ... }: {
  nixosConfigurations.bamos = nixpkgs.lib.nixosSystem {
    system = "x86_64-linux";
    modules = [
      bamos.nixosModules.default
      {
        # Môi trường Desktop
        bamos.desktop = "hyprland";
        
        # Driver NVIDIA sẵn sàng
        services.xserver.videoDrivers = [ "nvidia" ];
        
        # Ứng dụng cho người Việt
        environment.systemPackages = with pkgs; [
          unikey google-chrome wpsoffice
          zoom-us zalo vscode docker
        ];
      }
    ];
  };
};
