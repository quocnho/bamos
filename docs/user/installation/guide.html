📖 Hướng dẫn Cài đặt BamOS

BamOS hỗ trợ hầu hết PC hiện đại (x86_64, UEFI). Yêu cầu tối thiểu: 4GB RAM, 30GB ổ cứng, USB 8GB.

📹 Video hướng dẫn cài đặt chi tiết (sẽ cập nhật sau)

Bước 1: Tải ISO

Tải phiên bản BamOS phù hợp từ trang Download.

Bước 2: Tạo USB Boot

Sử dụng một trong các công cụ sau:

  • Balena Etcher — Khuyên dùng, giao diện đồ họa đơn giản
  • Rufus (Windows) — Chế độ DD Image
  • dd (Linux/Mac) — sudo dd if=bamos-*.iso of=/dev/sdX bs=4M status=progress

Bước 3: Boot từ USB

  • Khởi động lại máy, vào Boot Menu (F12/F2/F7/Del tùy hãng)
  • Chọn USB và boot
  • Chọn "BamOS Live" từ menu GRUB

Bước 4: Cài đặt

Trên màn hình Live, chạy trình cài đặt Calamares:

  1. Chọn ngôn ngữ (Tiếng Việt có sẵn)
  2. Chọn múi giờ (Asia/Ho_Chi_Minh)
  3. Chọn bàn phím (Vietnamese)
  4. Phân vùng ổ cứng — Disko tự động tạo:
    • Ổ C: Hệ thống, Nix store, config người dùng — có thể ghi đè khi cài lại
    • Ổ D: Documents, Downloads, Pictures, Videos — an toàn tuyệt đối
  5. Tạo tài khoản người dùng
  6. Nhấn "Cài đặt" và chờ hoàn tất

💾 Phân vùng Ổ C — Ổ D

BamOS tự động phân vùng ổ cứng theo mô hình quen thuộc:

SubvolumeMountVai tròKhi cài lại
@/Hệ thống, NixOS config📝 Ghi đè
@nix/nixNix store (tái sử dụng)♻️ Giữ lại
@home/homeConfig cá nhân♻️ Giữ lại
@data/data📁 Ổ D: Documents, Downloads, Pictures, Videos, Music✅ AN TOÀN

Khi cài lại BamOS, bạn có thể format lại Ổ C (hệ thống) mà không sợ mất dữ liệu trên Ổ D. Tất cả thư mục như Documents, Downloads, Pictures, Videos, Music đều được tự động trỏ sang /data.

Mẹo

Sau khi cài đặt, khởi động lại và gõ tiếng Việt ngay với Ctrl+Space!
