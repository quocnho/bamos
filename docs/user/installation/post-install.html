🔧 Hướng dẫn Sau Cài đặt

Sau khi cài đặt BamOS thành công, bạn nên thực hiện một số bước sau để hệ thống hoạt động ổn định và tối ưu nhất.

1. Cập nhật hệ thống

Ngay sau khi đăng nhập lần đầu, hãy mở terminal và chạy lệnh cập nhật:

# Cập nhật toàn bộ hệ thống
$ ublue-update --force

Lệnh này sẽ tự động tải xuống và áp dụng bản cập nhật BamOS mới nhất từ remote image.

Ghi chú

BamOS sử dụng cơ chế cập nhật atomic image-based. Sau khi cập nhật, bạn cần khởi động lại để chuyển sang deployment mới.

2. Cài đặt trình điều khiển (driver)

BamOS tự động nhận diện và cài đặt hầu hết driver. Tuy nhiên, với card đồ họa NVIDIA, bạn cần kích hoạt driver độc quyền:

# Kích hoạt driver NVIDIA
$ rpm-ostree override remove mesa-va-driver mesa-vdpau-driver
$ rpm-ostree install akmod-nvidia xorg-x11-drv-nvidia-cuda

Cảnh báo

Với laptop hybrid (Intel/NVIDIA Optimus), hãy dùng phiên bản BamOS có NVIDIA tích hợp sẵn thay vì tự cài driver thủ công.

3. Cài đặt ứng dụng cơ bản

Dùng GNOME Software (hoặc KDE Discover) để duyệt và cài ứng dụng Flatpak. Một số ứng dụng gợi ý:

  • Firefox — Trình duyệt web
  • LibreOffice — Bộ ứng dụng văn phòng
  • Thunderbird — Email client
  • GIMP / Inkscape — Đồ họa
  • VLC — Xem phim, nghe nhạc
  • Việc làm 🎮 — Steam, Heroic Games Launcher (Flatpak)

4. Cấu hình gõ tiếng Việt

BamOS đã tích hợp sẵn IBus-Bamboo. Bạn chỉ cần bật lên:

  1. Vào Cài đặt → Bàn phím → Nguồn vào
  2. Thêm "Vietnamese (Bamboo)"
  3. Sử dụng Ctrl+Space để chuyển đổi

5. Kết nối VPN (nếu cần)

Vào Cài đặt → Mạng → VPN và thêm cấu hình VPN của bạn. Xem thêm tại trang 🔒 VPN.

6. Tinh chỉnh hệ thống

BamOS có nhiều tùy chỉnh sẵn có. Tham khảo thêm tại:

  • ⚙ Tweaks — Các tinh chỉnh nâng cao
  • 🖥 Software Center — Hướng dẫn quản lý phần mềm
  • ⌨ CLI — Công cụ dòng lệnh BamOS

Mẹo

Bạn có thể dùng bazzite-rollback-helper để kiểm tra và quản lý các deployment nếu cần rollback.
